Olá! Este é o meu primeiro post no blog. Estou usando [Jekyll](https://jekyllrb.com/) para criar este site estático hospedado no GitHub Pages.

## O que é Jekyll?

Jekyll é um gerador de sites estáticos que transforma arquivos Markdown em HTML. É perfeito para blogs e sites pessoais porque:

- **Simples** — escreva em Markdown, sem banco de dados
- **Gratuito** — hospede no GitHub Pages sem custo
- **Rápido** — sites estáticos carregam muito mais rápido
- **Seguro** — sem servidor, sem vulnerabilidades

## Como criar um novo post

Para criar um novo post, basta criar um arquivo na pasta `_posts/` com o formato:

```
_posts/YYYY-MM-DD-titulo-do-post.md
```

Com o frontmatter no início:

```yaml
---
layout: post
title: "Título do Post"
date: 2026-04-10
categories: [blog]
---
```
